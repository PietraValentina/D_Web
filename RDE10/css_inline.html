CSS

Definição

Responsável pela estilização do site: cores, fontes, tamanho, sombras e posicionamentos. Bem como espaçamentos, alinhamentos, bordas entre outros.

Cascading Style Sheets - Folhas de Estilo em Cascata- tem por objetivo realizar a estilização de conteúdo de documentos HTML.

Componentes:

  • Seletores - Seleciona elementos HTML com base no nome do elemento ao qual o estilo será aplicado.
  • Propriedades - o aspecto específico do elemento que você quer usar como estilo.
  • Valores - a configuração aplicada ao atributo.
    •

    Sintaxe:

    seletor elemento {
    propriedades: valor; /*comentário*/
    propriedades - subpropriedades: valor;
    }

    Formas de utilizar o CSS

    Ordem cascata, ordem de prioridade.
    Inline>Incorporando>Externo.

    Inline

    Os códigos Css são definidos dentro de uma tag HTML Afetam somente a tag em que foi definida.

    Incorporado

    Os códigos CSS são definidos no cabeçalho, dentro da tag head. Afetam somente o documento em que foi definido.

    Externo

    Os códigos CSS são definidos em um arquivo externo. Afetam todos os documentos que estão linkados ao CSS.
